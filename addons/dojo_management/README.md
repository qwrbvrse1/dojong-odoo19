# Dojo Management

![cover](cover.png)

## Overview
Dojo Management is a comprehensive Odoo 19 module to manage students, belt rankings, martial arts styles, sessions, attendance, payments and advanced analytics for martial-arts schools.

## Features
- **Student Management** - Complete profiles with photos, belt ranks, and martial arts styles
- **Belt Ranking System** - Track progression from White Belt to Black Belt (5 Dan levels)
- **Martial Arts Styles** - Support for Karate, Taekwondo, Judo, Jiu-Jitsu, Kung Fu, and more
- **Interactive Dashboard** - Rich analytics with belt distribution, style breakdown, and KPIs
- **Session Scheduling** - Plan and manage training sessions
- **Attendance Tracking** - Batch attendance wizard with comprehensive history
- **Payment Management** - Track payments with late payment alerts
- **Belt Promotion Wizard** - Streamlined student rank advancement
- **Advanced Analytics** - Revenue tracking, attendance rates, and performance metrics
- **Customizable Access Rights** - Role-based security

## Installation
1. Place the module folder into your Odoo addons path.
2. Update app list and install **Dojo Management** from Apps.

## Usage
Once installed you'll find **Dojo Management** under the **Education** category:
1. Create students with photos, belt ranks, and martial arts styles.
2. Schedule sessions.
3. Run the attendance wizard.
4. Record payments.
5. Promote students using the belt promotion wizard.
6. View interactive dashboard with analytics.
7. Export comprehensive reports.

## Configuration
- Adjust access rights in `security/ir.model.access.csv`.
- Belt ranks are pre-configured but can be customized.
- Martial arts styles can be extended in the student model.
